# home-soc-lab

Домашняя лаборатория для практики навыков SOC-аналитика и 
анализа информационной безопасности.

🖥️ Среда

| Машина | ОС | IP | Роль |
|--------|----|----|------|
| Атакующий | Kali Linux | 192.168.56.102 | Симуляция атак |
| Цель | Windows 10 | 192.168.56.101 | Анализируемая система |
| SIEM | Wazuh (Ubuntu) | 192.168.56.103 | Мониторинг и алерты |

Платформа: VirtualBox, изолированная Internal Network

Write-up'ы

| 01 | [Сканирование сети и брутфорс SMB](./writeup-01-nmap-bruteforce.md) | nmap, Hydra, Wazuh |

Используемые инструменты

- **Wazuh SIEM** — мониторинг событий и алертов
- **Nmap** — сканирование сети и портов
- **Hydra** — симуляция брутфорс атак
- **Wireshark** — анализ сетевого трафика
